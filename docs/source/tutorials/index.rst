*********
Tutorials
*********
In these tutorials you will learn how to build a dsgrid project and datasets by using the
StandardScenarios project located at https://github.com/dsgrid/dsgrid-project-StandardScenarios.

In order to follow along please clone the repository locally. The steps assume that the respository
is stored at ``~/dsgrid-data``.


.. code-block:: console

   $ git clone https://github.com/dsgrid/dsgrid-project-StandardScenarios ~/dsgrid-data
